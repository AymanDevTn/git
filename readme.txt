# AymanDevTn Git Project

This project contains initial template files for a modern web development environment.

## Files included:
- index.html: A premium, responsive landing page.
- index.py: A basic Python initialization script.
- readme.txt: This project description file.

## How to run:
- Open index.html in any modern browser to view the design.
- Run `python index.py` to see the initialization output.
